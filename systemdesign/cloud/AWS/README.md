# Global vs Regional Services
## Global Services
- IAM ( Identity and Access Management)
- Route 53 (DNS Service)
- CloudFront( CContent Delivery Network)
- WAF ( Web application Firewall)

## Regional Services
- Amazon EC2 (IaaS)
- Elastic BeanStalk (PaaS)
- AWS Lambda (FaaS)
- Rekognition (SaaS)
